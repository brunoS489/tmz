# Publication du site

Ce dossier contient une version statique du site, prête à être publiée gratuitement.

## Option recommandée

GitHub Pages

1. Crée un dépôt GitHub public.
2. Mets le contenu de ce dossier à la racine du dépôt.
3. Vérifie que `index.html` est bien à la racine.
4. Active `Settings > Pages` sur la branche `main` et le dossier `/`.

## Alternative

Cloudflare Pages ou Netlify fonctionnent aussi avec exactement le même dossier.
